# Skill Creator

Use when the user wants to author or package new skills.

## Description

`skill-creator` is a situational skill for generating, testing, and publishing agent skills from within the agent ecosystem.

## When to use

- The user requests help building or packaging a new skill.
- The user asks how to make a skill compatible with the skills.sh ecosystem.
